# Simurgh: Song of Ink (Vertical Slice)

- Engine: Godot 4.x
- Run: Open this folder in Godot, run main scene `res://scenes/main.tscn`.
- Controls: WASD/Arrows to move, J attack, K dash, C hold-draw to cast (drag mouse), ESC pause.
- Mobile: Drag on screen to cast (basic support).

Export to Android: install Godot Android export templates, set your SDK/NDK, then create an export preset. This vertical slice uses only built-in shapes and particles.
